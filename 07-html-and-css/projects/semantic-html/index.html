My HTML Cheat Sheet

Headings

Headings from 'h1' to 'h6' identify blocks and sections. Every page must have one 'h1'.

Paragraphs

Paragraphs of text go in 'p' tags.

Lists

Unordered Lists

Unordered lists are marked with bullet points and go in 'ul' tags. Each list item goes in a 'li' tag.

Types of lists:

  • Unordered Lists ('ul')
  • Ordered Lists ('ol')

Ordered Lists

Ordered lists are numbered and go in 'ol' tags. Each list item goes in a 'li' tag.

Anchors (i.e. links)

Anchor tags create links to other pages. The URL goes in the 'href' attribute. Go to DuckDuckGo to search for more information.

Setting 'target="_blank"' will open the link in a new window. See MDN for more details.

Images

You can use the img tag to add images to your page.

To create a table, you start with the "table" element. By structuring your table with "table", "thead", "tbody", and "tfoot", you create a clear separation of the table's different sections. Inside the "tbody" "thead" and "tfoot" , you use "tr" elements to represent each data row, the 'scope' attribute can be used to indicate whether a "th" cell represents a column or row header, and within each row, you use "td" elements to hold the data cells.

Other contest section tags

  1. Article element:

    2. The 'article' element represents a self-contained composition in a document, page, application, or site, which is intended to be independently distributable or reusable

  2. Footer element:

    3. The 'footer' HTML element represents a footer for its nearest ancestor sectioning content or sectioning root element, typically contains information about the author of the section, copyright data or links to related documents.

  3. Header element:

    4. The 'header' element represents introductory content, typically a group of introductory or navigational aids. It may contain some heading elements but also a logo, a search form, an author name, and other elements.

  4. Nav element:

    5. The 'nav' element element represents a section of a page whose purpose is to provide navigation links, either within the current document or to other documents. Common examples of navigation sections are menus, tables of contents, and indexes.
